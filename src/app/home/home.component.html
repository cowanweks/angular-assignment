Home

Welcome to the Weather API service. This application provides a simple way to interact with weather data through various routes. Below is an overview of the available routes:

  • Home - The main page where you can find an overview of the Weather API and its functionalities.
  • API - This route provides detailed information about how to use the Weather API, including endpoints and request formats.
  • Form - This route includes a form where you can input specific parameters to fetch weather data directly from the API.
